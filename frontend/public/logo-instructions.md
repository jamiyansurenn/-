# Logo Setup Instructions

## ДААЦЫН ЦАМХАГ Logo

Please place your `ДААЦЫН ЦАМХАГ` logo PNG file in this directory with the exact name:

**File name:** `logo.png`

**Location:** `frontend/public/logo.png`

### Requirements:
- **Format:** PNG
- **Recommended size:** 120x120px or higher (will be scaled automatically)
- **Background:** Transparent preferred
- **File name:** Must be exactly `logo.png`

### Usage:
The logo will automatically appear in:
- **Header** - 60x60px (navigation bar)
- **Footer** - 50x50px (footer section)

### Fallback:
If the logo file is not found, a placeholder will be displayed automatically with the text "ДАЦ".
